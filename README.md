## About the game!
Roboquix is a simple game where you move Lil Robo by using the arrow keys or WAD (right key/D for right, left key/A for left, up key/W for jump; down key/S does nothing), towards the end portal. But in the process, you have to dodge saws and spikes, collect coins to open locked blocks blocking your way, adjust to physics-altering mechanics, and more!
If you complete a level fast enough, you’ll be rewarded with a medal accordingly, which helps increase your score, especially if you avoid dying while dodging obstacles.
